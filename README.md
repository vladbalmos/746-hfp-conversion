## British GPO 746 bluetooth conversion (HFP)

Firmware and hardware related resources to add hands-free bluetooth support to a 746 british phone - WIP.

## Hardware requirements

* Pico W
* 746 telephone
* pico probe (optional)

## Software dependencies

* pico-sdk (1.5.0)
* opencd (optional)
* pico probe (optional)


## Build

    make build
    
## Install
Via pico probe:

    make install
